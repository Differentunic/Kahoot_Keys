# Kahoot_Keys
This is a small program written in python that allows you to utilise the (<ins>J</ins> <ins>K</ins> <ins>N</ins> <ins>M</ins>) keys to answer questions on [Kahoot.it](https://kahoot.it)

### Usage:

- **J:** Top Left	
- **K:** Top Right
- **N:** Bottom Left
- **M:** Bottom Right
